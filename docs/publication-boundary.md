# Publication Boundary / 公开边界

This repository is a public, GitHub-ready archive of the Energia smart electronic scale prototype.

## Included

- `SmartElectronicScale/`: the Energia sketch, HX711 helper, and TimerA helper used by the sketch.
- `docs/state-machine.md`: a clean public state-machine summary rebuilt from the source code.
- `README.md`, `LICENSE`, and `.gitignore`: repository-facing documentation and hygiene files added during publication.

## Excluded

- Course task books and experiment reports from the local source folder.
- The original experiment state-diagram image/source from the local folder, because its note area contains personal student metadata.
- Any file that is not required to understand, build, or discuss the public firmware prototype.

The goal is to keep the public repository useful without publishing private identifiers or unrelated classroom documents.
